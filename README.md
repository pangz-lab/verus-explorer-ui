Prerequisite:
1. Node.js 
2. npm
3. grunt
3. bower

# Development
```bash
npm install
```

Install Grunt CLI globally
```bash
npm install -g grunt-cli
```

Install Bower globally
```bash
npm install -g bower
```

Install necessary dependencies via Bower:
```bash
bower install
```

Compile the project
```bash
grunt compile
```

Run to enable hotreload
```bash
grunt
```
